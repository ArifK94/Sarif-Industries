# Sarif Industries

A non-commerical E-commerce site in selling prosthetic products.

## Technologies
  - C#
  - MySQL
  - Linq
  - Ajax
  - Javascript
  - HTML
  - CSS

## Features

### Products
  - Display products
  - Search and filter products

### Shopping Cart
  - Add items to cart
  - Edit item in cart
  - Remove item from cart
  - Checkout from cart
    
### Account
  - Register & Login
  - Update profile
  - Edit profile
  - View orders
